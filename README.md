# 🏀 Basketball Scoreboard

A simple Basketball Scoreboard application built using HTML, CSS, and JavaScript.

## 🚀 Features

* Increase Home score (+1, +2, +3)
* Increase Guest score (+1, +2, +3)
* Reset both scores

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
